# Mini Social App

A simple social media application with user authentication, posting, and interaction features.

## Features
- User registration and login system
- Create and view posts in chronological feed
- Like posts with real-time counter
- Comment on posts with user attribution
- Persistent data storage using LocalStorage

## Tech Stack
- Frontend: HTML5, CSS, JavaScript
- Storage: LocalStorage (client-side)
- Authentication: Simple email/password system

## Usage
1. Open `index.html` in browser
2. Sign up with name, email, and password
3. Login to access the main app
4. Create posts, like, and comment

## Data Storage
- User accounts stored in LocalStorage
- Posts with likes and comments persist across sessions
- Current user session maintained for seamless experience
