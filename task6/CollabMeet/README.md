# CollabMeet - Video Conferencing Tool

A web-based video conferencing and collaboration platform with screen sharing, file sharing, and whiteboard features.

## Features
- Multi-user video conference layout (4 participants)
- Screen sharing capability
- File sharing interface
- Interactive whiteboard for collaboration
- Audio/video controls with mute functionality
- Responsive design for mobile and desktop

## Tech Stack
- Frontend: HTML5, CSS3
- Layout: CSS Grid and Flexbox
- Icons: Inline SVG graphics
- Responsive: Media queries for mobile/tablet/desktop

## Usage
1. Open `task6.html` in browser
2. View video grid layout for participants
3. Use control panel for mute, video, screen share
4. Access side panels for collaboration features

## Interface
- Main video grid (4 participant slots)
- Side panel with screen share, files, whiteboard
- Bottom control bar with meeting controls
- Fully responsive design
