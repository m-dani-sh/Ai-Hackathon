# QR Connect - Friend Connection Platform

A QR code-based friend connection platform that enables instant profile sharing and networking without usernames or phone numbers.

## Features
- Generate QR codes for user profiles
- Hover to reveal QR codes on friend cards
- Click to view enlarged QR codes with full profiles
- Simulated QR scanning functionality
- Responsive grid layout with modern dark theme
- Profile information including interests and contact details

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- QR Generation: qrcode-generator library
- Icons: Font Awesome 6.4.0
- Design: Modern dark theme with gold accents

## Usage
1. Open `index.html` in browser
2. Hover over friend cards to reveal QR codes
3. Click cards for detailed profile view
4. Use "Scan Friend QR" button for demo scanning
5. Connect with friends via simulated scanning

## Profile Data
Includes name, role, email, phone, location, interests, and profile images for professional networking connections.
