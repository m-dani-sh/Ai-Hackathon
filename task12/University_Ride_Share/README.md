# University Ride Share

A Flask-based ride-sharing platform for university students to post, search, and book rides.

## Features
- Post rides with route details and available seats
- Search rides by route
- Book available seats
- View ride history

## Tech Stack
- Backend: Flask (Python)
- Database: SQLite
- Frontend: HTML/CSS

## Setup
1. Install Flask: `pip install flask`
2. Run: `python app.py`
3. Open: http://localhost:5000

## Database
SQLite database with rides table storing:
- Driver info, route, time, seats
- Location coordinates
- Booking status
