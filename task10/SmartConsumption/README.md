# SwapShare Pakistan

A barter and rental platform for expensive items in Pakistan, enabling users to swap, rent, and share valuable goods.

## Features
- Browse items by category with search and filters
- Post items for barter or rental with pricing
- Smart matching algorithm for compatible trades
- User reviews and rating system
- Image upload and preview functionality

## Categories
Electronics, Vehicles, Property, Furniture, Mobiles, Jobs, Services, Other

## Tech Stack
- Frontend: HTML5, Bootstrap 5, JavaScript
- Storage: LocalStorage (client-side)
- UI: Bootstrap Icons, responsive design

## Usage
1. Open `index.html` in browser
2. Browse available items or add your own
3. Use smart matching to find compatible trades
4. Leave reviews for completed transactions

## Data
Items stored locally with user ratings, reviews, and availability status for barter/rental options.
