# Campus Resource Management System

A simple web-based system for booking and managing campus resources like labs, lecture halls, and projectors.

## Features
- Book resources with time slots
- Admin dashboard for managing requests
- Add new resources and time slots dynamically
- Approve/reject booking requests
- Conflict detection for double bookings

## Resources
Computer Labs, Lecture Halls, Projectors, and custom resources

## Tech Stack
- Frontend: HTML5, CSS, JavaScript
- Storage: In-memory (session-based)
- UI: Simple card-based layout

## Usage
1. Open `task9.html` in browser
2. Select resource and time slot to book
3. Admin can approve/reject requests
4. Add new resources and slots as needed

## Workflow
Users submit booking requests → Admin reviews → Approve/reject with notifications → Confirmed bookings stored
